## 개요
- **목표**: 1인 강사가 학생의 학습 영상을 업로드하면, AI가 주요 동작을 분석하여 정교한 피드백 시트를 자동 생성해 주는 서비스.
- **타겟**: 요가, 필라테스, 골프, 댄스 등 운동 강사 및 **농구, 배구 등 구기종목 코치**.

### 3. 주요 기능 (Features)
- **코치 대시보드**: 학생 관리, 영상 분석 히스토리, 성장 트렌드 확인.
- **AI 동작 분석**: 업로드된 영상의 스켈레톤 추출 및 핵심 물리 지표(각도, 대칭, 파워 등) 계산.
- **학생 포털 (NEW)**: 학생 전용 로그인/대시보드.
    - **내 피드백**: 코치가 보낸 정밀 분석 리포트 확인.
    - **추천 상점 (AI Shop)**: 분석 데이터 기반 맞춤형 건강식품/보충제 추천 (쿠팡/네이버 스토어 연동).
    - **프리미엄 레슨**: 개인별 맞춤형 유료 운동 가이드 영상 판매.
- **피드백 시트 공유**: 생성된 분석 결과를 학생에게 PDF 또는 링크로 즉시 공유.

### 4. 수익화 전략 (Monetization)
1.  **SaaS 구독**: 코치 대상 (무제한 분석, 학생 관리 툴).
2.  **제휴 마케팅 (Affiliate)**: 학생 앱 내 분석 기반 제품 추천 수수료 (쿠팡 파트너스 등).
3.  **유료 콘텐츠**: 학생 대상 프리미엄 운동 영상 및 루틴 판매.
4.  **PDF 리포트 요금**: 학생 소장용 고퀄리티 리포트 출력 요금.

## 수익화 모델
- **무료 플랜**: 월 3회 분석 무료.
- **프로 구독 (월 9,900원)**: 무제한 분석, 고화질 리포트 익스포트, 학생 히스토리 무제한 저장.
- **추가 패키지 (Pay-as-you-go)**: 분석 크레딧 10회권 별도 구매 가능.
