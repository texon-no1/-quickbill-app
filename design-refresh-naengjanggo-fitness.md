# naengjanggo-fitness – 디자인 리뉴얼 프롬프트

## 1. 프로젝트 개요

- 프로젝트 이름: `naengjanggo-fitness` (냉장고 파먹기 앱)  
- 저장소: `https://github.com/texon-no1/-quickbill-app`  
- 메인 파일: `naengjanggo-fitness.html`  
- 기술 스택:
  - Firebase Authentication (Google 로그인)
  - Firebase Firestore
  - Vanilla HTML/CSS/JS
  - GitHub Pages 호스팅 (`https://texon-no1.github.io/-quickbill-app/naengjanggo-fitness.html`)

---

## 2. 요청 목적

- 현재 `naengjanggo-fitness.html`의 UI는 **기능적으로 완성**되어 있으나,  
  - 디자인은 **현대적인 느낌**이 부족합니다.  
- **요즘 시대에 맞는, 여성 운동·피트니스 콘텐츠를 즐겨보는 사람**이 좋아할 만한  
  - **모던하고 깔끔한 UI/UX 디자인**을 요청합니다.  

---

## 3. 디자인 컨셉 (요청하는 방향)

### 3.1 타깃 사용자

- **20~30대 여성**  
  - 피트니스·운동·다이어트/식단/헬스 류 콘텐츠를 즐기는 사용자  
  - 레이아웃이 깔끔하고,  
  - 심미적으로 잘 정돈된 디자인을 선호하는 타입  
- **운동 부담감을 줄이고, “집에서 냉장고로 시작하는 피트니스”** 브랜딩을 강조

### 3.2 브랜딩 컬러 컨셉

- **마더 컬러**:  
  - `#F7F7F7` (배경)  
  - `#4A4A4A` (텍스트 메인)
- **메인 악센트 컬러**:  
  - `#4CAF50` (녹색) or `#FFD700` (노란색)  
  - 운동/다이어트/피트니스 이미지를 상징.
- **보조 컬러**:  
  - `#FF6B6B` (레드)  
  - `#4A90E2` (블루)  
  - 조합을 사용해 카드별로 강조 색상을 다르게 주는 방식도 가능.

### 3.3 레이아웃 구조

- **1페이지 형식**이지만,  
  - 레이아웃은 **모던한 블로그/앱 UI**처럼 구성  
- **헤더**:
  - `나의 냉장고 파먹기`  
  - 로그인/로그아웃 버튼  
- **바디**:
  - `나의 냉장고 재료` 섹션  
  - `추천 레시피` 섹션  
  - `추천 상품(네이버 쇼핑커넥트)` 섹션  
  - `출석/챌린지` 섹션
- **푸터**:
  - `회사 정보`, `문의`, `약관` 링크

### 3.4 디자인 스타일

- **스타일**:  
  - `Apple Reminders`/`Pinterest`/`Notion`/`Google Fit` 스타일의  
  - **깔끔하고, 미니멀한 UI**
- **폰트**:
  - `Roboto` 또는 `Inter` 같은 **모던 서체** 사용  
  - 헤더: `1.2em`  
  - 본문: `1em`  
- **버튼**:  
  - `Rounded(둥근 모서리)` + `Shadow(그림자)`  
  - `Primary`(녹색), `Secondary`(검정)로 나누기  
- **카드**:
  - 레시피/추천 상품/출석 정보는 **카드 형식**으로 나열  
  - `image + title + description + button` 구조

### 3.5 모바일 최적화

- **모바일**에서 **반응형**으로 레이아웃이 자동으로 조정  
  - `flex/grid` 기반으로 구성  
  - `max-width: 600px` 기준으로 레이아웃이 조정  
- **네비게이션**:
  - `navbar` 또는 `bottom tab` 형식으로  
  - `홈`, `냉장고`, `레시피`, `검색`, `프로필` 탭 구성

### 3.6 브랜드 요소

- `냉장고 파먹기` 브랜드는  
  - 냉장고 안에 있는 재료를 활용한  
  - **집에서 끝내는 피트니스** 컨셉  
- **아이콘**:
  - `냉장고 아이콘` + `운동/헬스 아이콘`을 조합한  
  - `네이티브 브랜드 아이콘`을 제안.
