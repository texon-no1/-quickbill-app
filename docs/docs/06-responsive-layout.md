# docs/06-responsive-layout.md

## 1. 목표

- 현재 CoachBuddy 웹 버전은 **태블릿/PC용으로 최적화**되어 있으나,  
  휴대폰에서 열면 레이아웃이 난잡하고 가독성이 떨어집니다.
- 목표:
  - **모바일(휴대폰)**: 심플하고 직관적인 세로형 레이아웃
  - **태블릿/PC**: 지금보다 더 넓고 풍부한 정보 제공

## 2. 화면 분류

- **모바일(휴대폰)**
  - 디바이스: 스마트폰 (가로: ~414px, 세로: ~896px)
  - 특징: 넓은 가로 공간 없이, 수직 스크롤에 의존

- **태블릿/PC**
  - 디바이스: 태블릿, 노트북, PC
  - 특징: 넓은 화면, 가로 공간 활용 가능

## 3. 모바일(휴대폰)용 레이아웃 요구사항

- 네비게이션: 상단에 **간단한 탭 메뉴** 또는 **하단 탭 바** 사용
- 카드형 레이아웃: 모든 정보를 세로 카드로 정렬
- 텍스트: 폰트 크기 확대, line-height 조정
- 버튼: 터치하기 편한 크기(적어도 44px 높이)
- 화면 전환: AJAX/SPA 스타일로, 페이지 이동 시 풀로드 방지

### 3.1. 페이지별 모바일 레이아웃

- `index.html` (코치 대시보드)
  - 상단: 코치 프로필, 학생 수, 분석 횟수 카드
  - 중간: 학생 리스트(카드형, 세로로 나열)
  - 하단: 메뉴 탭(대시보드, 학생, 리포트, 프리미엄)

- `student.html` (학생 포털)
  - 상단: 학생 프로필, 최근 분석 결과 카드
  - 중간: 그래프(세로 라인 차트, 텍스트로 요약)
  - 하단: 코칭 코멘트, 추천 루틴(카드형, 세로로 나열)

- `analysis.html` (분석 결과)
  - 상단: 분석 결과 요약(텍스트로 요약, 수치 강조)
  - 중간: 이미지(스켈레톤/운동 이미지)
  - 하단: AI 코멘트, 프리미엄 레슨/추천 상품 카드

---

## 4. 태블릿/PC용 레이아울 요구사항

- 네비게이션: 좌측 또는 상단 **메뉴 바** 사용
- 레이아웃: **그리드(2열, 3열)** 를 활용해 카드를 넓게 배치
- 사이드바: 메뉴, 통계, 최근 활동 표시
- 그래프: 훨씬 큰 차트, 자세한 통계
- 프리미엄 레슨: 카드 크기 확대, 썸네일 높이 증가

### 4.1. 페이지별 태블릿/PC 레이아웃

- `index.html`
  - 좌측: 메뉴 바(대시보드, 학생, 리포트, 프리미엄)
  - 우측 상단: 코치 프로필, 요약 카드
  - 우측 중앙: 학생 리스트(2열 그리드)
  - 우측 하단: 통계/그래프

- `student.html`
  - 좌측: 학생 메뉴 탐색, 학습 기록
  - 우측 상단: 최근 분석 카드
  - 우측 중앙: 그래프(큰 라인 차트)
  - 우측 하단: 코칭 코멘트, 프리미엄/추천 루틴

- `analysis.html`
  - 좌측: AI 분석 메타데이터, 메트릭 리스트
  - 우측: 이미지, 그래프, AI 코멘트, 프리미엄/추천 상품

---

## 5. CSS 구현 방향

- `style.css`에서 **Media Query**를 사용해 화면을 구분:
  - `@media (max-width: 768px)` : 모바일
  - `@media (min-width: 769px)` : 태블릿/PC
- 각 화면 크기별로
  - 그리드, 폰트 크기, 버튼 크기, 여백 등을 조정
- 카드(class="card") 요소는
  - 모바일: height: auto; width: 100%
  - 태블릿/PC: height: 200px; width: 200px; 또는 더 크게

## 6. 안티그래비티에게 작업 요청 예시

- 1) “style.css에 Media Query를 사용해 모바일/태블릿/PC 레이아웃을 정의해주세요.”  
- 2) “각 페이지(index.html, student.html, analysis.html)에 대해 모바일/태블릿/PC 레이아웃을 설계하고, HTML 구조를 구현해 주세요.”  
- 3) “카드형 레이아웃을 모든 페이지에 통일하고, 모바일에서는 수직, 태블릿/PC에서는 수평/그리드로 보이게 조정해주세요.”  

이렇게 하면,  
CoachBuddy는 모바일/태블릿/PC 어디서나 **일관된 UI/UX 경험**을 제공하게 됩니다.
